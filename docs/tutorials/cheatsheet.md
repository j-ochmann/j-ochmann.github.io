# CheatSheet

## 0️⃣ ShellGlue

### Git
```bash
clear &&
git add . &&
git commit -m "gc" &&
git push
```

### C
vyčištění obrazovky, kompilace s kontrolou a spuštění
```bash
clear &&
gcc -Wall -Wextra -Werror hello.c -o hello &&
./hello
```
