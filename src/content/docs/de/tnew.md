## Den Test ausführen

Nehmen Sie eine kleine Änderung an einer tschechischen .md-Datei im Ordner src/content/docs/cs/ vor und übertragen Sie sie auf GitHub.

Sehen Sie sich die Registerkarte „Aktionen“ an, um zu sehen, ob das Skript erfolgreich eine Verbindung zu Ihrem Server herstellt.

Änderung in der Datei

Sicherheits-Tipp: Zeigen Sie diesen Schlüssel nirgends öffentlich an (für Einrichtungszwecke ist er im Chat in Ordnung, aber belassen Sie ihn niemals im Code auf GitHub). GitHub Secrets verbirgt es, sodass es im Aktionsprotokoll nur als *** angezeigt wird. Die GitHub Secrets-Dokumentation bestätigt, dass dies der sicherste Weg für 2026 ist.
