[**Entwurfsmuster**](https://cs.wikipedia.org/wiki/N%C3%A1vrhov%C3%BD_vzor) in der Softwareentwicklung beschreibt eine wiederverwendbare Lösung für ein häufig auftretendes Problem in einem bestimmten Softwareentwurfskontext.[1] Ein Entwurfsmuster ist kein fertiger Entwurf, der direkt in Code übersetzt werden kann. Es handelt sich um eine Beschreibung oder Vorlage zur Lösung eines Problems, die in vielen verschiedenen Situationen verwendet werden kann.[2] Entwurfsmuster können als formalisierte Best Practices betrachtet werden, die ein Programmierer zur Lösung häufiger Software-Designprobleme verwenden kann.
