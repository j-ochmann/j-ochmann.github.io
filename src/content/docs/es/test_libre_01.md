## Ejecutando la prueba

Intente realizar un pequeño cambio en algún archivo checo .md en la carpeta src/content/docs/cs/ y envíelo a GitHub.

Mire la pestaña Acciones para ver si el script se conecta correctamente a su servidor.

"Consejo" de seguridad: no muestre esta clave en ningún lugar públicamente (está bien en el chat para fines de configuración, pero nunca la deje en el código de GitHub). GitHub Secrets lo ocultará para que solo se muestre como *** en el registro de acciones. La documentación de GitHub Secrets confirma que este es el camino más seguro para 2026.
